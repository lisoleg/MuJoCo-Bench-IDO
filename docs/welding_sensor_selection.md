> **用途**: 为MuJoCo-Bench-IDO焊接仿真系统提供物理传感器选型参考, 仿真中对应的传感器模型见 `core/welding_sensors.py`

## 详细选型

### 1. 电弧电流传感器

| 参数 | 规格 |
|------|------|
| 原理 | 霍尔效应闭环 |
| 量程 | ±400 A DC |
| 频率响应 | DC-100 kHz (-3dB) |
| 精度 | ±0.5% rdg ±0.1A |
| 工作温度 | -40~+85°C |
| 供电 | ±15V DC |
| 输出 | 0-5V / 4-20mA |
| 推荐型号 | LEM HASS 400-S |

**仿真对应**: `ArcCurrentSensor` — 基础采样率 50kHz, 自适应采样 η>1.0→4x

**Ψ-Anchor用途**: 回烧检测 (current > 350A AND voltage < 5V → critical)

### 2. 电弧电压传感器

| 参数 | 规格 |
|------|------|
| 原理 | 差分电压测量 |
| 量程 | 0-50 V DC |
| 频率响应 | DC-100 kHz |
| 精度 | ±0.2% rdg |
| 共模抑制 | >80 dB @ 1kHz |
| 工作温度 | -40~+85°C |
| 供电 | 5V DC |
| 输出 | 0-3.3V |
| 推荐方案 | 隔离运放 + 24-bit ADC |

**仿真对应**: `ArcVoltageSensor` — 基础采样率 50kHz

**Ψ-Anchor用途**: 回烧检测 + 电弧长度计算 (arc_length = V - 14)

### 3. 送丝速度传感器

| 参数 | 规格 |
|------|------|
| 原理 | 编码器反馈 |
| 量程 | 0-15 m/min |
| 精度 | ±1% rdg |
| 编码器分辨率 | 1024 ppr |
| 工作温度 | -10~+60°C |
| 供电 | 5V DC |
| 推荐方案 | 增量编码器 + 软件测速 |

**仿真对应**: `WireSpeedSensor` — 基础采样率 1kHz

### 4. 焊接速度传感器

| 参数 | 规格 |
|------|------|
| 原理 | 机器人TCP速度反馈 |
| 量程 | 0-20 mm/s |
| 精度 | ±0.5% rdg |
| 数据来源 | 机器人控制器 (6-DOF关节编码器) |
| 通信 | EtherCAT / CANopen |
| 更新率 | 500 Hz |

**仿真对应**: `TravelSpeedSensor` — 基础采样率 500Hz

### 5. 焊枪位置传感器 (TCP)

| 参数 | 规格 |
|------|------|
| 原理 | 机器人正运动学 |
| 量程 | ±500 mm (相对焊缝起点) |
| 精度 | ±0.1 mm |
| DOF | 6 (x, y, z, rx, ry, rz) |
| 数据来源 | 机器人控制器 |
| 更新率 | 500 Hz |

**仿真对应**: `TorchPositionSensor` — 基础采样率 500Hz, 输出6维TCP位姿

**用途**: 焊缝跟踪偏差计算 (TCP与waypoint的距离)

### 6. 温度传感器

| 参数 | 规格 |
|------|------|
| 原理 | K型热电偶 / 红外测温 |
| 量程 | 0-1500°C |
| 精度 | ±2°C (@1000°C) |
| 响应时间 | <100 ms |
| 工作温度 | -20~+1200°C (探头) |
| 供电 | 3.3V DC |
| 推荐方案 | K型热电偶 + MAX6675 (低成本) / 红外 MLX90614 (非接触) |

**仿真对应**: `TemperatureSensor` — 基础采样率 60Hz

**物理模型**: `T(t+1) = T(t) + heat_input × k_heat - k_cool × (T(t) - T_ambient)`

### 7. 焊缝跟踪传感器

| 参数 | 规格 |
|------|------|
| 原理 | 激光结构光 |
| 量程 | ±5 mm (横向偏差) |
| 精度 | ±0.05 mm |
| 测量频率 | 500 Hz |
| 工作距离 | 50-150 mm |
| 光源 | 660nm 红色激光, Class 2M |
| 通信 | GigE Vision / USB3 |
| 推荐型号 | KEYENCE LJ-X8000A / ScanTrack |

**仿真对应**: `SeamTrackingSensor` — 基础采样率 500Hz

**用途**: 实时焊缝偏差反馈, η残差计算的重要组成部分

## 数据采集架构

```
传感器层          采集层              处理层
┌──────┐        ┌──────────┐       ┌──────────────┐
│ 电流 │──50kHz─→│          │       │              │
│ 电压 │──50kHz─→│ FPGA     │──────→│ T-Proc       │
│ 送丝 │──1kHz──→│ 数据采集 │ EtherCAT│ η-ALU       │
│ 速度 │──500Hz─→│ 卡       │       │ Ψ-Anchor    │
│ TCP  │──500Hz─→│ (K26)    │       │ κ-Snap       │
│ 温度 │──60Hz──→│          │       │              │
│ 焊缝 │──500Hz─→│          │       │ EML蒸馏      │
└──────┘        └──────────┘       └──────────────┘
                                          │
                                     ┌────▼────┐
                                     │  Host   │
                                     │  CPU    │
                                     │ (VLA)   │
                                     └─────────┘
```

## 数据存储格式

焊接过程数据存储为 HDF5 格式:

```
/welding_session_001/
    /metadata/           — 会话元数据
    /sensors/
        /arc_current/    — (N,) float32, 50kHz采样
        /arc_voltage/    — (N,) float32, 50kHz采样
        /wire_speed/     — (N,) float32, 1kHz采样
        /travel_speed/   — (N,) float32, 500Hz采样
        /torch_position/ — (N, 6) float32, 500Hz采样
        /temperature/    — (N,) float32, 60Hz采样
        /seam_tracking/  — (N,) float32, 500Hz采样
    /timestamps/         — 各传感器的时间戳
    /ksnap_chain/        — κ-Snap审计链
    /eml_nodes/          — EML节点 (八元数表示)
```

## 注意事项

1. **电气隔离**: 所有传感器与焊接电源必须电气隔离, 推荐光耦/磁耦隔离
2. **EMI防护**: 焊接环境强电磁干扰, 传感器线缆使用双绞屏蔽线
3. **热管理**: 焊枪附近传感器需耐高温 (>200°C), 推荐陶瓷封装
4. **实时性**: η-ALU需要50kHz电流数据, 采集链路延迟 <20μs
5. **数据对齐**: 多传感器数据需在T-Proc中进行时间戳对齐 (η-ALU同步)
